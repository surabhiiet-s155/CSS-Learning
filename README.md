# 🎨 CSS Learning Journey (12 Days)

This repository documents my **12-day journey of learning CSS fundamentals**.  
Each day contains practice files and examples that helped me understand how to design and style web pages.

Through this project, I learned the core concepts of **CSS styling, layout design, Flexbox, and responsive design**.

---

# 📚 Topics Covered

During this learning journey, I practiced the following CSS concepts:

- Introduction to CSS
- Types of CSS (Inline, Internal, External)
- Colors, Backgrounds, and Borders
- Text Styling and Fonts
- CSS Box Model
- Display Property
- Position Property
- Flexbox Layout
- Form Styling
- Responsive Design using Media Queries
- Building Mini Web Projects

---

# 📅 Daily Progress

## Day 1 – Introduction to CSS
Learned the basics of CSS, types of CSS, and how to apply basic styling using selectors.

## Day 2 – Colors, Background & Borders
Applied text colors, background colors, borders, and border radius.

## Day 3 – Text Styling & Fonts
Practiced font-family, font-size, font-weight, text-decoration, and text spacing.

## Day 4 – CSS Box Model
Understood margin, padding, border, and how element spacing works.

## Day 5 – Display Property
Learned the difference between block, inline, and inline-block elements.

## Day 6 – Position Property
Explored static, relative, absolute, and fixed positioning.

## Day 7 – Flexbox (Part 1)
Introduced Flexbox layout using justify-content and align-items.

## Day 8 – Flexbox (Part 2)
Practiced vertical layouts and spacing using Flexbox.

## Day 9 – Form Styling
Styled input fields, buttons, and created a simple contact form.

## Day 10 – Mini Website Project
Built a simple webpage combining HTML structure and CSS layout.

## Day 11 – Responsive Design
Learned Media Queries and made layouts responsive for smaller screens.

## Day 12 – Portfolio Project
Created a simple personal portfolio webpage using HTML and CSS.
